# 🏭 SCM 협력사 변경 검토 대시보드

> **SCM팀 외주생산 파트 전용** — 협력사 변경 검토 보고서를 작성하고 관리하는 도구입니다.

---

## 📌 접속 방법

브라우저(Chrome 권장)에서 아래 링크로 접속하세요.

```
https://nanyounglee.github.io/scm_vendor_change/
```

> 별도 설치 없이 링크만으로 사용 가능합니다. 인터넷 연결이 필요합니다.

---

## 🗂️ 화면 구성

```
┌─────────────────────────────────────┐
│  ① 굿즈 선택    ② 파츠 선택         │  ← 검색 영역
├─────────────────────────────────────┤
│  ③ AS-IS 정보  (자동 입력)           │  ← 현재 협력사 현황
│  ④ TO-BE 정보  (직접 입력)           │  ← 변경 후 협력사 정보
├─────────────────────────────────────┤
│  ⑤ 수량별 단가 비교 차트             │
│  ⑥ 리드타임 비교                    │
│  ⑦ 담당자 의견                      │
├─────────────────────────────────────┤
│  ⑧ 저장된 보고서 아카이브            │
└─────────────────────────────────────┘
```

---

## 📝 보고서 작성 방법 (단계별)

### Step 1 — 굿즈 검색

- 상단 검색창에 **굿즈명 일부**를 입력하면 자동완성 목록이 나타납니다.
- 목록에서 해당 굿즈를 클릭해 선택합니다.

### Step 2 — 파츠 선택

- 굿즈를 선택하면 해당 굿즈의 파츠 목록이 드롭다운에 표시됩니다.
- 변경 검토 대상 파츠를 선택합니다.
- **파츠를 선택하는 순간 AS-IS 정보가 자동으로 채워집니다.**

### Step 3 — AS-IS 정보 확인

파츠 선택 시 아래 항목이 **자동 입력**됩니다.

| 항목 | 설명 |
|------|------|
| 현재 협력사명 | Airtable 파츠 데이터 기준 |
| MOQ | 최소 발주 수량 |
| 임가공 여부 | 해당 파츠의 임가공 여부 |
| 수량별 단가 | 1 / 50 / 100 / 300 / 500 / 1,000 / 3,000개 기준 |
| 리드타임 | 100 / 300 / 500 / 1,000 / 3,000ea 기준 (Phantom 파츠 기준) |
| 이슈 현황 | 최근 품질·납기·기타 이슈 목록 |
| 판매량 추이 | 연도별 판매 수량 차트 |

> AS-IS 항목은 **수동으로 수정 가능**합니다. Airtable 데이터와 실제 현황이 다를 경우 직접 수정하세요.

### Step 4 — TO-BE 정보 입력

변경 후 협력사 정보를 직접 입력합니다.

| 항목 | 입력 방법 |
|------|----------|
| 변경 협력사명 | 직접 텍스트 입력 |
| 변경 사유 | 드롭다운 선택 (품질/원가/납기/기타) |
| 수량별 단가 | 각 수량별로 직접 입력 (원 단위) |
| 리드타임 | 각 수량별로 직접 입력 (일 단위) |

### Step 5 — 비교 결과 확인

단가와 리드타임을 입력하면 AS-IS 대비 차이가 자동 계산됩니다.

- 🔴 **빨간색**: TOBE가 ASIS보다 불리 (단가↑ / LT↑)
- 🔵 **파란색**: TOBE가 ASIS보다 유리 (단가↓ / LT↓)

### Step 6 — 담당자 의견 작성

자유 텍스트로 검토 의견을 작성합니다. 변경 권고 여부, 리스크, 특이사항 등을 기재하세요.

### Step 7 — 보고서 저장

**[저장 및 아카이브]** 버튼을 클릭하면 보고서가 브라우저 로컬에 저장됩니다.

> ⚠️ 저장 데이터는 브라우저 로컬스토리지에 보관됩니다.
> 브라우저 캐시를 지우면 저장된 보고서가 삭제됩니다.
> 중요한 보고서는 반드시 MD 또는 PDF로 다운로드해 별도 보관하세요.

## 💾 보고서 다운로드

저장된 보고서는 두 가지 형식으로 내보낼 수 있습니다.

| 버튼 | 형식 | 용도 |
|------|------|------|
| **MD 다운로드** | `.md` (마크다운) | 옵시디언, Notion 등 문서 툴에 붙여넣기 |
| **PDF 다운로드** | `.pdf` | 보고·공유용 최종본 |

---

## 🗃️ 아카이브 관리

화면 하단 **아카이브** 섹션에서 저장된 보고서 목록을 확인할 수 있습니다.

- **불러오기**: 저장된 보고서를 다시 폼에 불러와 수정 가능
- **삭제**: 해당 보고서를 로컬스토리지에서 제거

---

## ⚠️ 주의사항

1. **Chrome 브라우저** 사용을 권장합니다 (PDF 출력 품질 최적화).
2. 데이터는 **접속한 기기의 브라우저에만 저장**됩니다. 다른 PC에서는 보이지 않습니다.
3. AS-IS 단가 및 리드타임 데이터는 **Airtable 데이터 기준**이며, 최신 현황과 다를 수 있습니다. 이상 시 파트장에게 문의하세요.
4. 리드타임은 **Phantom 파츠(추상적 산출물) 기준**으로 표시됩니다.
